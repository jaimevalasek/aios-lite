✅ Conclusão

O aios-lite v0.1.12 é uma alternativa substancialmente mais eficiente ao aios-core para a maioria dos casos de uso reais. A principal vantagem estrutural é o sistema de classificação por complexidade: um script ou landing page ativa 2 agentes; uma plataforma SaaS ativa 8. O aios-core ativa 12 agentes para qualquer projeto.


Com a incorporação da metodologia Akita nesta versão — execução atômica no @dev, protocolo de sessão no @orchestrator e spec.md como documento vivo — o lite ganhou o que faltava: disciplina de processo para projetos que duram semanas, sem adicionar peso de tokens para projetos curtos (spec.md é opcional).


A única vantagem real do aios-core hoje é para organizações grandes com processos muito maduros que precisam de granularidade extrema na gestão de histórias. Para todos os outros casos — desde scripts até plataformas SaaS — o aios-lite entrega mais rápido, com menos tokens e com processo suficientemente rigoroso.
