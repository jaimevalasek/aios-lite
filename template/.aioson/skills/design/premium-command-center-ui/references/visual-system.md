# Visual System

This package preserves the AIOS-style premium command-center language.

## Signature

- dark graphite foundations
- aurora-glass accents used sparingly
- borders-first depth
- compact density for operational reading
- restrained semantic color
- high contrast without neon overload

## Typography

- Prefer clean system grotesk or neutral premium sans
- Use mono only for operational metadata
- Keep headings firm and compact

## Surface rules

- At most three visible surface levels
- Borders separate more often than shadows
- One focal block per screen region

## Color behavior

- Accent should stay cool and controlled
- Status colors must remain semantic
- Avoid rainbow palettes and decorative gradients
