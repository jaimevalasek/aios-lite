# 51 - Scope Clarification: Future Optional Features

Date: 2026-03-01

## Decision
The following roadmap items are explicitly treated as future optional features:
- External Makopy MCP server
- Pipeline marketplace platform

## Current Priority
Focus on `aios-lite` core stability and expected behavior:
- reliable CLI flows
- cross-tool compatibility (Claude/Codex/Gemini/OpenCode)
- consistent i18n behavior
- robust test coverage and release quality

## Policy
- These optional features must not block current core releases.
- Core roadmap execution takes precedence until `aios-lite` is considered functionally complete.
